# 📊 HR Analytics Dashboard — Power BI

> Dashboard interativo para análise estratégica de indicadores de Recursos Humanos.

Este projeto demonstra a construção de um painel analítico no Power BI com foco em métricas essenciais para gestão de pessoas.  

O objetivo não é apenas visualizar dados, mas estruturar indicadores que apoiem decisões organizacionais relacionadas a quadro funcional, remuneração, engajamento e produtividade.

Este projeto representa o domínio de **Business Intelligence** dentro do engineering-portfolio.

---

## 🎯 Objective

Construir um dashboard analítico que permita:

- Visualização clara de KPIs de RH
- Modelagem de métricas com DAX
- Análise segmentada por função e gênero
- Interatividade via filtros dinâmicos
- Exploração de indicadores estratégicos

O foco está em transformar dados operacionais em informação executiva.

---

## 🧠 Analytical Structure

O dashboard foi estruturado com base em perguntas de negócio orientadas à tomada de decisão.

### Fluxo Analítico
Raw Dataset
↓
Modelagem de Dados
↓
Criação de Medidas (DAX)
↓
KPIs e Visualizações
↓
Interação do Usuário



---

## 📌 Business Questions Answered

O painel responde às seguintes perguntas estratégicas:

1. Qual o total de funcionários atualmente na empresa?
2. Qual o tempo médio de experiência do quadro funcional?
3. Qual o total e percentual de colaboradores por gênero?
4. Qual a média salarial mensal?
5. Qual a distribuição de funcionários por função?
6. Qual o percentual disponível para realização de horas extras?
7. Qual o nível médio de envolvimento no trabalho?

Essas perguntas estruturam os principais indicadores do dashboard.

---

## 📈 Key Metrics (KPIs)

- Total de Colaboradores
- Média Salarial
- Tempo Médio de Experiência
- Percentual por Gênero
- Distribuição por Cargo
- Percentual de Disponibilidade para Hora Extra
- Índice de Envolvimento

## ⚙️ How to Use

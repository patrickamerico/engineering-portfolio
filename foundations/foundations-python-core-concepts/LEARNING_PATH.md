# 📚 Learning Path — Foundations in Python

Este documento organiza os projetos deste repositório em uma trilha progressiva de aprendizado.

O objetivo é evoluir da lógica básica até conceitos estruturais mais organizados, como leitura de dados e orientação a objetos.

---

# 🟢 Level 1 — Fundamentos da Linguagem

Foco:

- Sintaxe básica
- Entrada e saída de dados
- Operadores matemáticos
- Estruturas condicionais
- Estruturas de repetição

## 📌 Projeto

### 🔹 basic_calculator.py

Consolida:

- Uso de `input()`
- Conversão de tipos
- Estruturas `if/elif/else`
- Operadores matemáticos
- Organização simples com funções

Objetivo do nível:
Dominar o fluxo básico de execução em Python.

---

# 🟡 Level 2 — Lógica Aplicada e Controle de Fluxo

Foco:

- Loops
- Validação de entrada
- Geração de números aleatórios
- Controle de estado simples

## 📌 Projeto

### 🔹 number_guessing_game.py

Consolida:

- Uso do módulo `random`
- Estrutura `while`
- Controle de tentativas
- Comparações condicionais
- Interação contínua com o usuário

Objetivo do nível:
Desenvolver raciocínio lógico aplicado e controle de fluxo mais elaborado.

---

# 🟠 Level 3 — Programação Orientada a Objetos (OOP)

Foco:

- Classes
- Métodos
- Encapsulamento
- Organização de estado interno

## 📌 Projeto

### 🔹 hangman_game_oop.py

Consolida:

- Criação de classes
- Definição de métodos
- Organização do código orientada a objetos
- Separação de responsabilidades
- Manipulação de strings

Objetivo do nível:
Introduzir organização estrutural e mentalidade de design de software.

---

# 🔵 Level 4 — Manipulação de Dados

Foco:

- Leitura de arquivos
- Processamento de dados estruturados
- Iteração sobre registros
- Organização de informações externas

## 📌 Projeto

### 🔹 csv_reader.py

Consolida:

- Uso do módulo `csv`
- Abertura e leitura de arquivos
- Iteração sobre dados tabulares
- Processamento básico de informações

Objetivo do nível:
Aproximar o aprendizado de cenários reais de dados e aplicações práticas.

---

# 🟣 Próximos Passos Recomendados

Após concluir todos os níveis, recomenda-se evoluir para:

- Tratamento de exceções com `try/except`
- Modularização em múltiplos arquivos
- Criação de pacotes
- Uso de ambientes virtuais
- Testes automatizados (pytest)
- Criação de APIs simples
- Integração com banco de dados

Esses tópicos permitem avançar para:

- Backend development
- Data engineering
- Automação profissional
- Arquitetura de software

---

# 🧠 Estratégia de Estudo Recomendada

1. Execute cada projeto.
2. Modifique partes do código.
3. Adicione pequenas melhorias.
4. Reescreva do zero sem consultar.
5. Explique a lógica em voz alta.

A consolidação real acontece quando você consegue explicar e reconstruir a solução.

---

# 🎯 Objetivo Final da Trilha

Ao concluir essa trilha você deverá dominar:

- Controle de fluxo
- Estruturas fundamentais do Python
- Organização de código
- Introdução à orientação a objetos
- Manipulação básica de dados externos

---

> Fundamentos sólidos tornam a evolução para sistemas complexos muito mais natural.

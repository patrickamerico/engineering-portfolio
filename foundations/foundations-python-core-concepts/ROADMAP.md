# 🗺️ Roadmap — Foundations Python Core Concepts

Este roadmap define a evolução planejada do repositório.

O objetivo é transformar este projeto em uma base sólida e progressiva de fundamentos em Python, preparando para backend, data engineering e arquitetura de software.

---

# ✅ Fase 1 — Consolidação dos Fundamentos (Atual)

Status: Concluído

Escopo:

- Sintaxe básica
- Condicionais
- Loops
- Funções
- Manipulação de strings
- Introdução à orientação a objetos
- Leitura de arquivos CSV

Projetos atuais:

- basic_calculator.py
- number_guessing_game.py
- hangman_game_oop.py
- csv_reader.py

Objetivo:
Estabelecer domínio da lógica de programação e estrutura fundamental da linguagem.

---

# 🔄 Fase 2 — Estrutura e Organização

Status: Próxima etapa

Melhorias planejadas:

- Aplicar tratamento de exceções (try/except)
- Modularizar código em múltiplos arquivos
- Separar responsabilidades em módulos
- Criar estrutura de pacotes Python
- Introduzir padrão main guard (`if __name__ == "__main__"`)

Objetivo:
Evoluir de scripts simples para estrutura organizada de aplicações.

---

# 🧪 Fase 3 — Qualidade e Testabilidade

Planejamento:

- Introduzir testes automatizados com pytest
- Criar funções puras para facilitar testes
- Adicionar validações robustas
- Melhorar mensagens de erro
- Aplicar princípios básicos de clean code

Objetivo:
Ensinar boas práticas desde a base.

---

# 📦 Fase 4 — Aplicações Mais Realistas

Expansões futuras:

- Projeto de CRUD simples em terminal
- Persistência em arquivo JSON
- Simulação de mini API local
- Pequeno sistema de cadastro modular
- Logging estruturado

Objetivo:
Simular cenários mais próximos do mercado.

---

# 🏗️ Fase 5 — Integração com Conceitos Avançados

Evoluções possíveis:

- Introdução a ambientes virtuais
- Uso de requirements.txt
- Introdução a tipagem com type hints
- Estrutura orientada a pacotes
- Padrões básicos de arquitetura

Objetivo:
Criar ponte para projetos backend e data engineering.

---

# 🔮 Visão de Longo Prazo

Este repositório pode evoluir para:

- Foundations → Core → Intermediate → Advanced
- Base para APIs REST
- Base para automações
- Base para pipelines de dados
- Base para sistemas backend completos

---

# 📌 Princípios do Roadmap

Este projeto seguirá sempre:

- Simplicidade antes de complexidade
- Clareza antes de abstração
- Evolução progressiva
- Fundamentos sólidos antes de frameworks

---

# 🎯 Meta Final

Transformar este repositório em:

- Referência pessoal de fundamentos
- Material estruturado para revisão
- Base didática para mentoring
- Pilar inicial do engineering-portfolio

---

> Fundamentos fortes reduzem drasticamente a complexidade dos próximos níveis.

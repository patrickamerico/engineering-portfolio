# 🧠 Core Concepts — Foundations in Python

Este documento descreve os principais conceitos de programação e fundamentos da linguagem Python abordados neste repositório.

O objetivo não é apenas implementar pequenos programas, mas consolidar a base necessária para evoluir para backend, automação, ciência de dados e engenharia de software.

---

# 📌 1. Sintaxe e Estrutura da Linguagem

Python é uma linguagem:

- Interpretada
- De tipagem dinâmica
- De alto nível
- Multi-paradigma

Características importantes:

- Indentação obrigatória
- Código legível
- Estrutura simples e expressiva

Este repositório reforça a escrita clara e organizada.

# 🐍 Foundations — Python Core Concepts

> Coleção de projetos práticos desenvolvidos em Python para consolidação de fundamentos da linguagem, lógica de programação e princípios de orientação a objetos.

Este repositório reúne exercícios e miniaplicações com foco em:

- Sintaxe e estrutura básica do Python
- Controle de fluxo
- Funções
- Manipulação de dados
- Leitura de arquivos
- Programação orientada a objetos (OOP)

O objetivo é construir uma base sólida para evolução em áreas como backend, automação, ciência de dados e engenharia de software.

---

# 🎯 Objetivo do Repositório

Consolidar os principais fundamentos da linguagem Python por meio de pequenos projetos práticos e progressivos.

Competências desenvolvidas:

- Estruturas condicionais (`if`, `elif`, `else`)
- Estruturas de repetição (`for`, `while`)
- Funções
- Manipulação de strings
- Listas e dicionários
- Leitura de arquivos CSV
- Programação orientada a objetos

# 📌 Descrição dos Arquivos

## 🔢 basic_calculator.py

Calculadora simples em linha de comando que realiza operações básicas:

- Soma
- Subtração
- Multiplicação
- Divisão

Conceitos trabalhados:

- Entrada de dados (`input`)
- Conversão de tipos
- Condicionais
- Funções

---

## 🎯 number_guessing_game.py

Jogo de adivinhação de número gerado aleatoriamente.

Conceitos trabalhados:

- Geração de números aleatórios
- Loops de repetição
- Controle de tentativas
- Validação de entrada
- Lógica condicional

---

## 🔤 hangman_game_oop.py

Implementação do jogo da forca utilizando Programação Orientada a Objetos.

Conceitos trabalhados:

- Classes e objetos
- Métodos
- Encapsulamento básico
- Controle de estado do jogo
- Manipulação de strings

Este arquivo introduz conceitos estruturais importantes para aplicações maiores.

---

## 📊 csv_reader.py

Leitura e manipulação de dados a partir de arquivos CSV.

Conceitos trabalhados:

- Leitura de arquivos
- Uso do módulo `csv`
- Iteração sobre dados estruturados
- Organização de informações tabulares

Esse exercício aproxima o aprendizado de cenários reais de análise e processamento de dados.

# 🛠 Requisitos

- Python 3.8 ou superior

Verifique sua versão com:

```bash
python --version
```

🚀 Como Executar

Execute qualquer arquivo individualmente:

```bash
python nome_do_arquivo.py
```
Exemplo:

```bash
python basic_calculator.py
```
Não é necessário instalar dependências externas.

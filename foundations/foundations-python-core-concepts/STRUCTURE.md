# 🧠 Arquivos de Código

Cada arquivo `.py` é:

- Independente
- Executável isoladamente
- Focado em um conceito específico
- Projetado para aprendizado progressivo

---

## 🔹 basic_calculator.py

Responsabilidade:

- Demonstrar entrada e saída de dados
- Aplicar operadores matemáticos
- Utilizar estruturas condicionais
- Organizar lógica básica em funções

Nível: Fundamentos iniciais

---

## 🔹 number_guessing_game.py

Responsabilidade:

- Trabalhar com geração de números aleatórios
- Implementar loops de repetição
- Validar entradas
- Controlar estado do jogo

Nível: Lógica aplicada e controle de fluxo

---

## 🔹 hangman_game_oop.py

Responsabilidade:

- Introduzir Programação Orientada a Objetos
- Criar classes e métodos
- Organizar estado interno
- Aplicar encapsulamento básico

Nível: Estrutura e organização orientada a objetos

---

## 🔹 csv_reader.py

Responsabilidade:

- Ler arquivos externos
- Processar dados estruturados
- Iterar sobre registros
- Trabalhar com dados tabulares

Nível: Manipulação de dados

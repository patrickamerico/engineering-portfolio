# 📚 Learning Path — Foundations in C

Este documento organiza os projetos deste repositório em uma trilha progressiva de aprendizado.

A ideia é evoluir da lógica básica até pequenas simulações estruturadas, consolidando os fundamentos da linguagem C.

---

# 🟢 Level 1 — Lógica Básica e Operações Fundamentais

Foco:
- Entrada e saída
- Operadores aritméticos
- Estruturas condicionais
- Estruturas de repetição simples

## 📌 Projetos

### 🔹 basic_calculator.c
Consolida:
- Operadores matemáticos
- Uso de `if` e `switch`
- Entrada via `scanf`
- Estrutura básica de programa

---

### 🔹 random_number_generator.c
Consolida:
- Uso de `rand()`
- Uso de `srand()`
- Controle de intervalos numéricos
- Conceito de pseudoaleatoriedade

# 🔵 Level 4 — Próximos Passos (Evolução Recomendada)

Após dominar os projetos atuais, recomenda-se evoluir para:

- Modularização com múltiplos arquivos `.c` e `.h`
- Uso de ponteiros
- Alocação dinâmica com `malloc` e `free`
- Leitura e escrita em arquivos
- Estruturas de dados como:
  - Listas encadeadas
  - Pilhas
  - Filas

Esses tópicos ampliam a base para:

- Programação de sistemas
- Desenvolvimento embarcado
- Estruturas de dados avançadas
- Linguagens como C++, Rust e Go

---

# 🧠 Estratégia de Estudo Recomendada

1. Compile e execute cada programa.
2. Modifique pequenas partes do código.
3. Adicione novas funcionalidades simples.
4. Reescreva o código sem consultar a versão original.
5. Explique o funcionamento em voz alta (consolidação cognitiva).

---

# 🎯 Objetivo Final da Trilha

Ao concluir todos os níveis, você deverá ter domínio de:

- Controle de fluxo
- Organização procedural
- Estruturas básicas de dados
- Modelagem simples
- Manipulação de memória estática
- Lógica algorítmica sólida

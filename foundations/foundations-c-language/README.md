# 🧠 Foundations — C Programming Language

> Coleção de projetos práticos desenvolvidos em C para consolidação de fundamentos de programação, lógica e estruturas básicas.

Este repositório reúne exercícios e miniaplicações desenvolvidas com foco em:

- Estruturas de controle
- Funções
- Manipulação de strings
- Arrays e matrizes
- Geração de números aleatórios
- Organização modular
- Simulação de pequenos sistemas

O objetivo é demonstrar domínio da base da linguagem C e fundamentos de Ciência da Computação.

---

## 🎯 Objetivo do Repositório

Consolidar fundamentos essenciais de programação utilizando C, incluindo:

- Lógica condicional (`if`, `switch`)
- Estruturas de repetição (`for`, `while`, `do while`)
- Manipulação de arrays
- Funções
- Entrada e saída padrão
- Uso básico da biblioteca padrão
- Organização de código procedural

## 🚀 Como Compilar e Executar

### Compilar um arquivo:

```bash
gcc nome_do_arquivo.c -o programa
```
Exemplo:
```
gcc basic_calculator.c -o calculator
```
Executar:
Linux / macOS:
```
./calculator
```
Windows:
```
calculator.exe
```

# 🎨 Product UI/UX Prototypes

Este repositório reúne protótipos de interfaces e experiências de usuário (UI/UX) desenvolvidos com foco em design de produto, usabilidade e pensamento estratégico.

O objetivo é demonstrar capacidade de transformar problemas em soluções visuais funcionais, aplicando princípios de design centrado no usuário.

---

# 🎯 Objetivo do Projeto

- Criar protótipos de interfaces para produtos digitais
- Explorar fluxos de navegação
- Validar ideias antes do desenvolvimento
- Aplicar princípios de usabilidade e design visual
- Documentar decisões de produto

Este projeto foca em **pensamento de produto e experiência do usuário**, não apenas em estética.

---

# 🧩 Tipos de Protótipos

Os protótipos podem incluir:

- 📱 Aplicações mobile
- 💻 Aplicações web
- 📊 Dashboards
- 🛒 E-commerces
- 🏥 Sistemas internos
- 🧠 MVPs de novos produtos

# 🛠️ Ferramentas Utilizadas

- Figma (principal)

- Miro (para fluxos e brainstorming)

- Ferramentas de wireframe e prototipação

# 🧠 Processo Aplicado

Cada protótipo pode seguir etapas como:

1. Definição do problema
2. Identificação do público-alvo
3. Mapeamento de jornada do usuário
4. Criação de wireframes
5. Design de alta fidelidade
6. Definição de componentes
7. Validação de fluxo

---

# 🧭 Conceitos de UX Aplicados

- Acessibilidade básica

- Redução de fricção

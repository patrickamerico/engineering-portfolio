# 🗂️ Frontend Task Management App

Aplicação frontend para gerenciamento de tarefas (Task Management), desenvolvida com foco em organização de estado, manipulação dinâmica do DOM e boas práticas de estruturação de código.

O objetivo do projeto é consolidar fundamentos de desenvolvimento frontend moderno, simulando um sistema simples de controle de tarefas.

---

# 🎯 Objetivo do Projeto

Construir uma aplicação de gerenciamento de tarefas que permita:

- Criar tarefas
- Editar tarefas
- Marcar como concluídas
- Remover tarefas
- Organizar visualmente o estado das tarefas

Este projeto reforça conceitos essenciais de frontend antes da adoção de frameworks avançados.

---

# 🚀 Funcionalidades

- ✅ Adicionar nova tarefa
- ✏️ Editar tarefa existente
- ✔️ Marcar tarefa como concluída
- 🗑️ Excluir tarefa
- 📋 Listagem dinâmica de tarefas
- 🔄 Atualização automática da interface
- 💾 Persistência local (se aplicável via LocalStorage)

---

# 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Manipulação de DOM
- LocalStorage (opcional)

# ▶️ Como Executar

1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. A aplicação estará pronta para uso.

Não é necessário backend ou instalação de dependências.

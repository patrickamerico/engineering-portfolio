# 🎵 Frontend UI Clone — Spotify

Este projeto consiste na recriação da interface do Spotify com foco exclusivo no desenvolvimento frontend.

O objetivo é praticar estruturação de layouts complexos, organização visual, responsividade e boas práticas de HTML e CSS, simulando uma interface real de produto amplamente utilizado.

---

# 🎯 Objetivo do Projeto

Reproduzir visualmente a interface do Spotify para consolidar:

- Estruturação semântica de HTML
- Layouts com Flexbox e/ou Grid
- Organização visual profissional
- Componentização visual
- Responsividade
- Atenção a detalhes de UI

Este projeto tem foco em **interface (UI)**, não em funcionalidades reais de streaming.

---

# 🚀 Funcionalidades Implementadas

- 🎧 Barra lateral de navegação
- 🎼 Área principal com playlists e destaques
- ▶️ Player fixo na parte inferior
- 🔍 Campo de busca (visual)
- 📱 Layout responsivo (se aplicável)
- 🎨 Estilização inspirada na identidade visual do Spotify

---

# 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid (se aplicável)
- Responsividade com Media Queries
- JavaScript (opcional para interações visuais)

1. Clone o repositório.
2. Abra o arquivo `index.html` no navegador.
3. Navegue pela interface.

Não é necessário backend ou instalação de dependências.
